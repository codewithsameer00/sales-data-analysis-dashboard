# Sales Data Analysis Dashboard

An interactive analytics dashboard that analyzes sales data and generates insights about revenue trends, product performance, and regional sales distribution.

## Features

- Revenue trend analysis
- Product performance insights
- Regional sales breakdown
- Interactive charts and dashboards
- Business intelligence insights

## Tech Stack

Frontend
- React
- Tailwind CSS
- Recharts

Backend
- Python
- FastAPI
- Pandas
